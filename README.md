# 质检训练营全自动化 Agent 系统

基于 OpenClaw Agent 框架 + DeepSeek Chat 构建的外呼中心质检自动化系统。

## 核心痛点
技术支持二中心(中文2组/7组、英文1组,约100人)每日需人工完成6项重复操作,耗时30-60分钟/天,频繁出错。

## 技术架构
- **框架**: OpenClaw Agent
- **推理引擎**: DeepSeek Chat
- **前端对接**: 飞书开放平台(消息/表格/多维表格/云文档/联系人)
- **触发方式**: Cron定时(13:00/13:30/19:00)+群聊@实时触发

## 核心逻辑(5步长链推理)
1. 文件定位:识别群聊Sheet/Wiki/Base链接,调API读取质检记录
2. 多维过滤:排除非目标组别,识别当通为零/当月为零/虚假三类扣罚
3. 入池推理(8分支决策树):英文队列当通为零及以上直接入池;中文队列需14天累计>=2次或当月为零/虚假入池
4. 批量写入:历史扣罚表+在池表,写入后回读校验三重重试
5. 报告生成:按模板填充并群发

## 实际成效
- 上线时间:2026年4月8日
- 处理数据:23份日报,1150+条记录,46条精准扣罚
- 出入池管理:中文队列14人次、英文队列6人次
- 效率提升:30-60分钟/天→0分钟自动化,降本100%
- 准确率:100%,零漏判零误判

## 目录说明
- `project_config.json` - 项目配置(时间规则、组别过滤、出入池定义)
- `message_templates.json` - 飞书群聊消息模板(13:00/13:30/预检结果/19:00执行)
- `AI_WORKFLOW_ENHANCED.md` - Agent工作流增强说明
